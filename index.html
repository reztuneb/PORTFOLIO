LAURENZ WEEKE

Use the `images/` folder for high-quality photos and the `videos/` folder for video files. Embed them directly in this page using simple HTML.

Embed an image

Place the file in images/, then use:

<img src="images/example.jpg" alt="Description" />

Embed a video

Place the file in videos/, then use:

<video controls playsinline muted loop>
  <source src="videos/example.mp4" type="video/mp4" />
</video>
