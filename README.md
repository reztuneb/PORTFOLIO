# Portfolio Site

This is an extremely simple portfolio website for Laurenz Weeke.

It contains a single page with a white background and the name **LAURENZ WEEKE** in bold Helvetica-style text.

## Media support

- Use `images/` for high-quality image files.
- Use `videos/` for video files.
- Both folders are served from the repo root, so you can reference them directly in `index.html`.

### Example image embed

```html
<img src="images/example.jpg" alt="Description" />
```

### Example video embed

```html
<video controls playsinline muted loop>
  <source src="videos/example.mp4" type="video/mp4" />
</video>
```

## Deployment

- Host on GitHub Pages from the repository root.
- Use `CNAME` to configure the custom domain: `laurenzweeke.com`.
